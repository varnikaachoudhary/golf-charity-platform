Golf Charity Subscription Platform (MVP)

This project is a frontend-based Minimum Viable Product (MVP) of a Golf Charity Subscription Platform designed to combine sports engagement with charitable contributions. The platform allows users to simulate a subscription-based experience where they can track their golf performance, participate in draw-based rewards, and support a charity of their choice.

The application provides a simple and interactive user dashboard where users can log in, select a preferred charity, and enter their golf scores. A key feature implemented is the score management system, which maintains only the latest five scores by automatically removing the oldest entry when a new score is added. This ensures efficient data handling and aligns with real-world application logic.

Additionally, the platform includes a simulated monthly draw system that generates random winning numbers, allowing users to experience the reward mechanism conceptually. User data such as login information, selected charity, and scores are stored using browser local storage to maintain persistence across sessions.

The user interface is designed to be clean, modern, and user-friendly, focusing on usability and engagement. While this implementation focuses on frontend functionality, it lays a strong foundation for future enhancements such as backend integration, secure authentication, payment processing, and an administrative dashboard.
